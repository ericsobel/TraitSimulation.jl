TraitSimulation.jl

A Julia module to perform trait simulation.

Installation

TraitSimulation requires Julia 0.5. To install TraitSimulation please type the following code in a Julia REPL:

Pkg.clone("https://github.com/huwenboshi/TraitSimulation.jl.git")

Uninstallation

To uninstall TraitSimulation please type the following code in a Julia REPL:

Pkg.rm("TraitSimulation")

For a complete uninstallation, please also remove the TraitSimulation directory in the .julia folder.

Key Features

  1. Simulate trait under Generalized Linear Model (GLM) or Generalized Linear Mixed Model (GLMM)
  2. Allow simulation of multiple correlated traits
  3. Support integration with SnpArrays
  4. Provide convenient ways to specify the simulation model
